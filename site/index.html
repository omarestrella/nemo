macOS menu bar for Dokku

Keep an eye on your Dokku apps from the menu bar.

Nemo pairs a native macOS app with a small read-only Linux agent on your Dokku host, so status, URLs, HTTPS state, logs, and platform events are always close by.

Install

Two pieces, one pairing flow.

macOS app

Install the signed release build with the Homebrew cask from the Nemo repo.

brew tap omarestrella/nemo https://github.com/omarestrella/nemo
brew install --cask omarestrella/nemo/nemo

Dokku host agent

Run the Linux installer on the Dokku host to install the agent and systemd service.

curl -fsSL https://omarestrella.github.io/nemo/install.sh | sh

Security model

Read-only by default.

The long-running service runs as a dedicated nemo-agent user. Root is only needed for install and repair. Paired clients use bearer credentials, and untrusted remote access should go through HTTPS or Tailscale Serve.
