Nemo
GitHub Releases

A menu bar helper for Dokku.

Nemo is a small Mac app with a Linux agent for a personal Dokku host. Check app status, URLs, HTTPS, logs, and recent events from the menu bar.

Install GitHub

Install.

Install the Mac app, run the agent installer on your Dokku host, then pair from Nemo.

macOS app

Install the signed app with the Homebrew cask from this repo.

brew tap omarestrella/nemo https://github.com/omarestrella/nemo
brew install --cask omarestrella/nemo/nemo

Dokku host agent

Run the installer on the host to install the agent and systemd service.

curl -fsSL https://omarestrella.github.io/nemo/install.sh | sh

What it does.

  • Shows app status. Running state, domains, and HTTPS at a glance.
  • Opens useful context. App URLs, logs, recent events, and host settings.
  • Pairs from the browser. Start in Nemo, approve on the host, and keep the credential in Keychain.
